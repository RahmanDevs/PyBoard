# PyBoard V2.1.5 (Beta)

A **Bangla keyboard layout** based on the **ANSI V1 font**.

## Features

- **ANSI Mode**: Traditional Bangla keyboard layout (default)
- **Unicode Mode**: Standard Unicode Bangla input
- **System Tray Integration**: Easy access from the taskbar
- **Application-Specific Settings**: Remembers your keyboard preference for each application
- **Hotkey Support**: Quick keyboard shortcuts for toggling

## Keyboard Shortcuts

| Shortcut | Function |
|----------|----------|
| **F1** | Toggle language (English ‚Üî Bangla) |
| **Shift+F1** | Toggle mode (ANSI ‚Üî Unicode) |
| **Ctrl+Shift+Q** | Quit application |

## How to Use

### From System Tray (Right-Click Menu)

1. **Right-click** the tray icon to open the menu
2. Click on the language option to toggle between:
   - Bangla (ANSI) ‚Üí English
   - Bangla (Unicode) ‚Üí English
   - English ‚Üí Bangla (ANSI/Unicode)
3. Click "Mode: ANSI [OFF]" or "Mode: Unicode [ON]" to toggle between ANSI and Unicode modes

### From Keyboard

- Press **F1** to switch between English and Bangla
- Press **Shift+F1** to switch between ANSI and Unicode modes

### Tray Icon Indicators

- **EN.ico** - English mode active
- **BN.ico** - Bangla (ANSI) mode active
- **BN_U.ico** - Bangla (Unicode) mode active
